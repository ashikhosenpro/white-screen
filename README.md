# White Screen

A simple, fast, and distraction-free full-screen white screen website. Perfect for lighting, screen testing, reading, photography, presentations, and more.

## ✨ Features

- Full-screen white display
- One-click fullscreen mode
- Fast and lightweight
- Responsive on desktop, tablet, and mobile
- No ads or distractions
- Works in all modern browsers

## 🚀 Use Cases

- 💡 Use as a Light
- 🖥️ Check Your Screen
- 📖 Read in the Dark
- 📷 Product Photography
- 🎤 Presentations & Screen Sharing

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript

## 📦 Getting Started

### Clone the repository

```bash
git clone https://github.com/ashikhosenpro/white-screen.git
```

### Open the project

```bash
cd white-screen
```

Then open `index.html` in your preferred web browser.
